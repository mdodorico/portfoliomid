Proyectos

Escuela de arte

https://github.com/mdodorico/escueladearte

Proyecto personal desarrollado en Java, HTML, CSS, Bootstrap y MySQL con Eclipse y MySQL Workbench. Los usuarios pueden registrarse, se verifica la identidad por email y pueden recuperar la contraseña. Cuenta además con un ABM de obras de arte, un formulario para contacto y roles para usuarios registrados. Las contraseñas se guardan encriptadas en la BD.

Librería

https://github.com/mdodorico/libreria

Proyecto final para la materia Programacion GUI II desarrollado en ASP.NET, HTML y MySQL (aplicando el patrón MVC) con Visual Studio y Microsoft SQL SMS. Contiene un ABM de productos, ABM usuarios y ABM proveedores, roles de usuario y un carro de compras totalmente funcional. Las contraseñas se guardan encriptadas en la BD.

Mi viaje

https://github.com/mdodorico/miviaje

Proyecto final de la materia Programación Web II desarrollado en Java, HTML, CSS y Bootstrap con Spring Boot y MySQL Workbench. Contiene ABM de propiedades en alquiler, ABM de categorias (tipo de propiedad), ABM de usuarios y roles de usuario. Las contraseñas se guardan encriptadas en la BD.

Prestame un libro

https://github.com/mdodorico/prestameunlibro

Proyecto final para la materia Programación Web I desarrollado en JavaScript y PHP con Xampp y phpMyAdmin. Cuenta con un ABM de usuarios, ABM de autores y ABM de libros, edición del perfil personal del usuario, servicio de mensajes entre usuarios registrados y servicio de ayuda mediante el envío de mensajes al Administrador del sitio.

Pandemia

https://github.com/mdodorico/pandemia

Proyecto final para la materia Programación II desarrollado en C++ con CodeBlocks. Es de tipo survival (el juego termina cuando el jugador se queda sin vidas), cuenta con 6 niveles con distintos mapas que van cambiando en la medida que el jugador suma puntos y los enemigos van aumentando en cantidad y velocidad de disparos. Para el desarrollo visual utilicé la librería Allegro.
